# Run the agents live in the app on your own computer (full, detailed steps)

Goal: see the REAL CrewAI agents answer inside the app. This runs entirely on your
machine, costs only a few cents of API usage, and needs no AWS.

You will keep ONE window open running a small server, then open the app.

-------------------------------------------------------------------------------
STEP 0 - What you need
-------------------------------------------------------------------------------
- Python 3.10, 3.11, or 3.12 already installed (you ran the crew earlier, so you have it).
- Your OpenAI API key (starts with sk-...). The same one you used before.

-------------------------------------------------------------------------------
STEP 1 - Open a command window
-------------------------------------------------------------------------------
Press the Windows key, type  cmd  , press Enter. A black "Command Prompt" window opens.
(Command Prompt is easier than PowerShell for this.)

-------------------------------------------------------------------------------
STEP 2 - Check Python works
-------------------------------------------------------------------------------
Type this and press Enter:

    python --version

You should see something like  Python 3.12.x .
If it says "not recognized", install Python from https://www.python.org/downloads/
and during install TICK the box "Add python.exe to PATH", then reopen cmd.

-------------------------------------------------------------------------------
STEP 3 - Go to the crew folder
-------------------------------------------------------------------------------
Copy-paste this whole line and press Enter:

    cd "C:\Users\yasha\OneDrive\Documents\Claude\Projects\USAII Hackathon\aristotle-decision-crew"

The start of the line should now show that folder.

-------------------------------------------------------------------------------
STEP 4 - Install the three packages (one time, takes a few minutes)
-------------------------------------------------------------------------------
Paste and press Enter:

    python -m pip install "crewai[tools]" fastapi "uvicorn[standard]"

Let it finish. Lots of text scrolling is normal. Wait for the prompt to return.
(If you already installed crewai earlier, it will just add fastapi and uvicorn.)

-------------------------------------------------------------------------------
STEP 5 - Give it your API key (for this window only)
-------------------------------------------------------------------------------
Paste this, replace sk-REPLACE with your real key, press Enter:

    set OPENAI_API_KEY=sk-REPLACE

Nothing visible happens - that is correct. The key lives only in THIS window.

-------------------------------------------------------------------------------
STEP 6 - Start the server
-------------------------------------------------------------------------------
Paste and press Enter:

    python -m uvicorn server:app --port 8000

Wait until you see a line like:

    Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)

LEAVE THIS WINDOW OPEN. This is your server. Closing it stops the agents.

-------------------------------------------------------------------------------
STEP 7 - Quick check that the server is alive
-------------------------------------------------------------------------------
Open your web browser and go to:

    http://localhost:8000/

You should see:  {"ok":true,"service":"aristotle decision crew"}
If you see that, the server is working.

-------------------------------------------------------------------------------
STEP 8 - Open the app (the local copy, endpoint already set)
-------------------------------------------------------------------------------
In your project folder, double-click:

    Aristotle-Decision-Coach_LOCAL-TEST.html

It opens in your browser. (This copy already points at http://localhost:8000/review.
Your main file Aristotle-Decision-Coach_CHALLENGE3.html stays offline-ready.)

-------------------------------------------------------------------------------
STEP 9 - See the live agents
-------------------------------------------------------------------------------
1. Click  "Try a sample decision".
2. Click through to Step 4 (the Reason / results screen).
3. Open the panel  "AI second opinions".
   It first says  "Calling the live CrewAI agents..."  then after about
   10-30 seconds it fills with real, model-generated blind spots, tradeoffs,
   and a premortem. The note changes to "Live, generated by the CrewAI agents."

That is the full app working end to end: app -> your server -> CrewAI -> back.

-------------------------------------------------------------------------------
STEP 10 - Stop when done
-------------------------------------------------------------------------------
Click the server window and press  Ctrl + C . Close the window.

===============================================================================
IF SOMETHING GOES WRONG
===============================================================================
- "pip is not recognized"            -> use  python -m pip install ...  (as written above).
- "python is not recognized"         -> reinstall Python, tick "Add to PATH", reopen cmd.
- App says "Live agents unavailable" -> look at the server window for a red error.
   Usually a wrong/empty key (redo Step 5) or no API credit on the key.
- "Address already in use" / port busy-> run with  --port 8010  instead, then in the
   LOCAL html change 8000 to 8010 (or just ask me to regenerate it).
- Panel stays on "Calling..."        -> the model is slow or the key has no quota;
   check the server window. The app falls back to the on-device version on failure.
- Browser shows a CORS error         -> make sure the URL is exactly
   http://localhost:8000/review  and the server window is still running.

Once this works locally, hosting it on AWS App Runner gives the same thing with a
public URL and no terminal - see README.md.
